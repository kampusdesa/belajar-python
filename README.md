### How to run this tutorial 
(without Anaconda)
1. Install pipenv (e.g; using brew)
```bash
$ brew install pipenv
```
2. Clone this repo
```bash
$ git clone git@github.com:kampusdesa/belajar-python.git
```
3. Go to `belajar-python` folder
```bash
$ cd belajar-python
```
4. Install python dependency packages
```bash
$ pipenv install
```
5. Run Jupyter notebook
```bash
$ jupyter notebook
```
